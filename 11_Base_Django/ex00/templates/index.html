Markdown Cheatsheet

Markdown est un langage de balisage léger qui permet de formater du texte de manière simple et lisible. Il est souvent utilisé pour rédiger de la documentation, des messages sur les forums, etc.
Voici une liste des éléments de syntaxe les plus courants en Markdown et leur rendu en HTML.

paragraphe 1

Les sauts de paragraphe se font en laissant une ligne vide entre les paragraphes.
