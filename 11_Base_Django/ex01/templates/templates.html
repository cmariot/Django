Moteur de template

Blocs :

Les blocs permettent de définir des zones dans un template qui peuvent être remplacées par d'autres contenus dans des templates enfants. Voici un exemple :

{% block contenu %}{% endblock %}

Boucles for ... in :

Les boucles for ... in permettent de parcourir des listes ou des dictionnaires dans un template. Voici un exemple :

{% for item in liste %}
    {{ item }}
{% endfor %}

Structures de contrôle if :

Les structures de contrôle if permettent d'effectuer des tests conditionnels dans un template. Voici un exemple :

{% if condition %}
    Contenu affiché si la condition est vraie.
{% else %}
    Contenu affiché si la condition est fausse.
{% endif %}

Affichage de variables passées par contexte :

Les variables passées par contexte depuis la vue peuvent être affichées dans un template en utilisant des balises doubles accolades. Voici un exemple :

<h1>{{ titre }}</h1>

Ceci représente un aperçu simplifié du fonctionnement du moteur de templates de Django, mettant en avant les fonctionnalités des blocs, des boucles for...in, des structures de contrôle if, et de l'affichage de variables passées par contexte.
